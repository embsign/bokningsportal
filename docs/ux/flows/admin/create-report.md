# Flow: Skapa debiteringsunderlag

Syfte: Skapa en CSV‑rapport för debiteringsunderlag.

1. **Start**
   - Admin klickar **Skapa rapport** i sektionen Debiteringsunderlag / Rapporter.

2. **Välj månad**
   - Admin väljer månad (t.ex. 2026‑03).

3. **Välj bokningsobjekt**
   - Admin väljer bokningsobjekt i dropdown.

4. **Ladda hem som CSV**
   - Admin klickar **Ladda ner CSV**.

Regler:
- Enkel trestegsprocess.
- CSV laddas ner direkt efter sista steget.
