# Flow: Redigera användare

Syfte: Låta admin hitta en användare och redigera alla fält i en modal.

1. **Start**
   - Admin klickar **Redigera** i sektionen Användare.

2. **Välj användare**
   - Lista över alla användare i systemet.
   - Sökfält för snabb åtkomst.
   - Admin väljer en användare i listan (Välj‑knapp).

3. **Redigera användare (popup)**
   - Popup/modal med möjlighet att redigera alla fält.
   - Fält:
     - Identitet
     - Lägenhets ID
     - Hus/Trapphus
     - RFID‑tagg
     - Behörighetsgrupper (Välj‑popup)
     - Admin (checkbox)
     - Status (Aktiv/Inaktiv)
   - Knappar: **Spara**, **Avbryt**

Regler:
- Alla fält ska vara redigerbara i popupen.
- Avbryt stänger utan att spara.
