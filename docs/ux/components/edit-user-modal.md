# Komponenten: Edit User Modal

Syfte: Redigera en användares fält i en modal.

## Fält
- Identitet
- Lägenhets ID
- Hus/Trapphus
- RFID‑tagg
- Behörighetsgrupper (Välj‑popup)
- Admin (checkbox)
- Status (Aktiv/Inaktiv)

## Interaktion
- **Spara** uppdaterar användaren.
- **Avbryt** stänger utan att spara.

Se även: `docs/design/components.md` (EditUserModal).
