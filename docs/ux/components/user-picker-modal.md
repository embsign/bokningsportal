# Komponenten: User Picker Modal

Syfte: Låta admin välja vilken användare som ska redigeras.

## Layout
- Modal med sökfält och lista över användare.
- Varje rad visar identitet + metadata (lägenhet, hus, grupper).

## Interaktion
- Sök filtrerar listan.
- **Välj** öppnar EditUserModal för vald användare.
- **Stäng** stänger modalen.

Se även: `docs/design/components.md` (UserPickerModal).
