# Komponenten: Report Modal

Syfte: Skapa och ladda ner debiteringsunderlag som CSV.

## Steg
1) Välj månad  
2) Välj bokningsobjekt  
3) Ladda ner CSV

## Interaktion
- **Nästa** kräver vald månad och bokningsobjekt.
- **Ladda ner CSV** laddar ner filen direkt.

Se även: `docs/design/components.md` (ReportModal).
